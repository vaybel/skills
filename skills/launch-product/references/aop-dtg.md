# AOP and DTG Guidance

## DTG

Use DTG for front/back graphic apparel ideas:

- tees
- hoodies
- sweatshirts
- tote bags
- simple chest, back, or centered poster graphics

Good DTG prompts mention composition, print placement, palette, and texture.

## AOP

Use AOP when the user asks for:

- all-over print
- repeating pattern
- full garment coverage
- cut-and-sew
- sublimation
- edge-to-edge artwork

Pass `--technique aop` so catalog selection uses AOP-capable blanks.

Good AOP prompts describe repeat structure, scale, motifs, spacing, and whether
the pattern should feel dense, sparse, tiled, or organic.
